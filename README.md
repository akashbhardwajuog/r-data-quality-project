# R Data Quality and Exploratory Analysis Project 
 
## Project objective 
To develop a reproducible R workflow for importing, cleaning, checking and visualising a public biomedical dataset. 
 
## Skills demonstrated 
- Data import and cleaning 
- Data-quality checks 
- Missing-data assessment 
- Exploratory data analysis 
- Statistical visualisation 
- Reproducible reporting 
- Version control using Git/GitHub 
 
## Dataset 
To be added once selected. 
 
## Workflow 
1. Import raw data 
2. Check data structure and variable types 
3. Identify missing values and duplicate records 
4. Clean and document data-processing decisions 
5. Produce descriptive statistics and visualisations 
6. Generate a reproducible report 
7. Document findings and limitations 
 
## Reproducibility 
The analysis will be completed in R/RStudio. Code will be stored in the `scripts` folder, outputs in the `outputs` folder, and reports in the `reports` folder.
